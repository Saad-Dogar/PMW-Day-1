LAYER 00 · HARDWARE

            
ONYX-16

            

              A fully emulated 16-bit computer, built from scratch in C++.
            

          

          

            
    
              
  • 
                Full fetch → decode → execute → writeback CPU pipeline with 8
                general-purpose registers, an ALU, and a 16-byte direct-mapped
                cache
              
    • 
              
  • 
                Custom two-pass assembler compiles ONYX assembly — written in
                English or Roman Urdu — straight to machine words
              
    • 
              
  • 
                Simulated power supply enforces a real 20W budget; the CPU
                thermal-halts at 90°C
              
    • 
              
  • 
                Boots into an OS menu, flashes compiled binaries into virtual
                RAM, and runs them cycle-by-cycle on a 32×16 phosphor display
              
    • 
            

            

              
onyx asm — bilingual

              
BHARO R0, myVar      ; load address
PARHO R1, R0         ; read value
JAMA  R1, R1         ; R1 = R1 + R1
AARAM                ; halt

            

          

          

            

              C++17OOPNo dependencies

LAYER 01 · OPERATING SYSTEM

            
The Aracanum Editor

            

              A multi-tab text editor built directly on raw Win32 — no rich-edit
              control.
            

          

          

            
    
              
  • 
                Up to 10 simultaneous tabs, each with its own buffer, cursor,
                alignment, and page state
              
    • 
              
  • 
                Custom Wrapper/Layout engine reflows the raw character buffer
                into lines, columns, and paginated pages on every render
              
    • 
              
  • 
                Incremental find with live cursor-jump, full clipboard
                integration, and four text alignment modes
              
    • 
              
  • 
                Every line, cursor, and selection is drawn manually with GDI —
                no third-party UI library
              
    • 
            

            

              
core shortcuts

              
Ctrl+Nnew tab

              
Ctrl+Ffind

              
Ctrl+L/R/E/Jalign

              
Ctrl+Ptoggle panel

LAYER 02 · APPLICATION

            
Tumble-Pop Replica

            

              A faithful recreation of the arcade classic, built in C++ with
              SFML.
            

          

          

            
    
              
  • 
                Vacuum up ghosts and skeletons, then fire them back as rolling
                projectiles that chain into other enemies
              
    • 
              
  • 
                Two playable characters, four powerups, and two structurally
                distinct levels — one with randomized layouts
              
    • 
              
  • 
                Independent movement, gravity, and animation state machines for
                each enemy type
              
    • 
              
  • 
                Sprites, audio, and input all handled through SFML — zero
                external game-engine dependencies
              
    •

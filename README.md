# Saad Dogar — Portfolio

## Projects featured

**Layer 00 — Hardware**
[ONYX-16](https://github.com/Saad-Dogar/ONYX-16) — a 16-bit virtual computer in C++17: CPU, ALU, RAM, bus, and a custom two-pass assembler with English/Roman-Urdu instruction mnemonics. Built from scratch, no external dependencies.

**Layer 01 — Operating System**
[The Aracanum Editor](https://github.com/Saad-Dogar/text-editor) — a multi-tab text editor built directly on the raw Win32 API. No rich-edit control, no third-party UI — every pixel rendered manually via GDI.

**Layer 02 — Application**
[Tumble-Pop Replica](https://github.com/Saad-Dogar/TumblePop-Game) — a faithful C++/SFML recreation of the arcade classic, with a vacuum-and-fire mechanic, chaining projectiles, and a powerup system.

**Layer 03 — Web**
[Weather Tracker](https://github.com/Saad-Dogar/weather-tracker) — a live weather dashboard in vanilla HTML/CSS/JS, with a 5-day forecast, auto geolocation, and dark/light mode. → [Live demo](https://saad-dogar.github.io/weather-tracker/)

# Documentation of 3D Reconstruction Experiments

This directory contains the training data, iteration checkpoints, and source assets utilized to explore modern 3D scene reconstruction methods for the **PreserveMy.World** initiative.

## Directory Structure

*   **/3DGS/**: Contains the overlapping image dataset and point cloud baseline configuration for 3D Gaussian Splatting testing.
*   **/NeRF/**: Contains training iterations and PSNR performance evaluations for Neural Radiance Fields.

## Core Objective
The purpose of these experiments is to benchmark reconstruction speed, geometric fidelity, and browser-side rendering capability to build a scalable digital heritage documentation pipeline.
